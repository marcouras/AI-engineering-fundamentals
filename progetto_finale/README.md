# Progetto Finale

> 🚧 *Cartella riservata al progetto finale — da completare entro la Lezione 6*

## Struttura consigliata

```
progetto_finale/
├── app.py              # App Streamlit
├── chatbot.py          # Logica chatbot
├── rag.py              # Pipeline RAG
├── tools.py            # Tool definitions
├── data/               # Documenti da indicizzare
├── requirements.txt
└── README.md           # La tua documentazione
```

## Il tuo README deve includere

1. **Descrizione** — cosa fa il chatbot, a chi è rivolto
2. **Stack** — tecnologie usate
3. **Come usarlo** — istruzioni di installazione
4. **Link deploy** — URL su Streamlit Cloud
5. **Crawl-Walk-Run** — dove si posiziona il progetto

## Idee di progetto

- Chatbot per il turismo in Sardegna
- Assistente per studenti ITS
- Chatbot per un'azienda locale
- Supporto tecnico su manuale utente via RAG
- Chatbot per studiare una materia specifica
